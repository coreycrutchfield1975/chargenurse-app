VA Charge RN Workstation

Individualized patient worksheet app for the Charge RN on a VA nursing home floor.

## Files
- index.html — Full app (all views, logic, forms)
- styles.css — 12 background themes + layout
- vercel.json — Static site deployment config

## Deploy
1. Go to https://vercel.com/new
2. Drag this folder onto the page
3. Name it chargenurse-workstation
4. Click Deploy
